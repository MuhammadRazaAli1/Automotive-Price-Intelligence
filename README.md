# 🚗 Automotive Price Intelligence - Real-Time Used Car Price Estimator

UniCarPrice is an end-to-end Machine Learning project that predicts the market value of used cars based on their specifications and text descriptions.

## 🌟 Features
* **Advanced ML Pipeline:** Utilizes a `RandomForestRegressor` embedded within a robust `scikit-learn` pipeline.
* **NLP Integration:** Processes listing titles and descriptions using `TfidfVectorizer` to capture hidden text features.
* **Real-time Web App:** An interactive, user-friendly UI built with `Streamlit`.
* **Automated Preprocessing:** Handles missing values, encodes categorical variables, and scales numerical features seamlessly.

## 🛠️ Tech Stack
* **Python 3.8+**
* **Machine Learning:** `scikit-learn`, `pandas`, `numpy`
* **Frontend:** `Streamlit`
